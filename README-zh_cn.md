# Choloepus

## 介绍
Choloepus 是一个使用三相电机驱动的FOC柔性夹爪

*实物图*

## 性能参数

- 最大抓力：可以夹起 10kg 的水
- 最小抓力：可以夹针操作，可以夹起蛋壳
- 最大驱动电流：2A
- 驱动电压：12V ~ 24V
